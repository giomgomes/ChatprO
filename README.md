## Pré-requisitos

Certifique-se de ter o Python 3.7 ou superior instalado em seu ambiente. 

## Configuração

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/giomgomes/ChatprO.git
   cd ChatprO

2. Instale as dependências:
As dependências do projeto estão listadas no arquivo requirements.txt. Instale-as usando o seguinte comando:

   ```bash
   pip install -r requirements.txt

3. Configure o arquivo "config.json":
Substitua as informações do arquivo config.json pelas infos do seu IBM Watson e Telegram

## Execução
Para executar o bot, use o seguinte comando:

   ```bash
   python main.py
